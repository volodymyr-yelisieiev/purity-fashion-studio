# PURITY Fashion Studio - Content Management Guide

This guide explains how to manage your website content through the Payload CMS admin panel.

## üîê Accessing the Admin Panel

1. Go to your website URL + `/admin` (e.g., `https://purity.studio/admin`)
2. Log in with your admin credentials
3. You'll see the dashboard with all content types

## üìù Managing Content

### Services

Services are the styling packages you offer to clients.

**To add a new service:**
1. Click **Services** in the sidebar
2. Click **Create New**
3. Fill in the required fields:
   - **Title** - Service name (e.g., "Personal Styling")
   - **Slug** - URL path (auto-generated from title)
   - **Category** - Select from: Styling, Atelier, Consulting, Shopping, Events
   - **Excerpt** - Short description for cards and previews
   - **Description** - Full service description
   
**Optional fields:**
- **Pricing** - Set prices in UAH and EUR
- **Featured** - Show on homepage
- **Bookable** - Allow online booking

**Language support:**
- Switch between EN / UK / RU tabs to add translations
- Content should be added for all languages

---

### Portfolio

Portfolio showcases your transformation work with before/after images.

**To add a portfolio item:**
1. Click **Portfolio** in the sidebar
2. Click **Create New**
3. Upload:
   - **Before Image** - Client before transformation (required)
   - **After Image** - Client after transformation (required)
4. Fill in:
   - **Title** - Project name
   - **Category** - Type of transformation
   - **Description** - Project details

**Optional:**
- **Gallery** - Additional process/detail images
- **Testimonial** - Client quote with name and rating
- **Featured** - Show on homepage

---

### Collections (Lookbooks)

Collections are curated fashion lookbooks.

**To add a collection:**
1. Click **Collections** in the sidebar
2. Click **Create New**
3. Fill in:
   - **Name** - Collection name
   - **Season** - Spring, Summer, Autumn, Winter, or All Season
   - **Description** - Collection story and inspiration
   - **Cover Image** - Main collection image
   
**Adding lookbook images:**
1. In the **Images** section, click **Add Image**
2. Upload or select from media library
3. Add optional caption for each image

**Linking products:**
- Use **Linked Products** to connect shop items to the collection

---

### Products

Products are items available in your online shop.

**To add a product:**
1. Click **Products** in the sidebar
2. Click **Create New**
3. Fill in:
   - **Name** - Product name
   - **Category** - Dresses, Tops, Bottoms, etc.
   - **Pricing** - Price in UAH (required), EUR (optional)
   - **Images** - Upload product photos

**Product details:**
- **Material** - Fabric composition
- **Care** - Washing/care instructions
- **Sizes** - Available sizes
- **Colors** - Color options with hex codes

---

### Courses

Educational programs and masterclasses.

**To add a course:**
1. Click **Courses** in the sidebar
2. Click **Create New**
3. Fill in:
   - **Title** - Course name
   - **Category** - Personal Styling, Color Analysis, etc.
   - **Level** - Beginner, Intermediate, Advanced
   - **Duration** - Length in hours/days/weeks
   - **Pricing** - Course price

---

## üñºÔ∏è Media Library

All images and files are stored in the **Media** library.

**Uploading images:**
1. Click **Media** in the sidebar
2. Click **Create New**
3. Drag & drop or click to upload
4. Add **Alt Text** for accessibility

**Best practices:**
- Use descriptive alt text for SEO
- Optimize images before upload (max 2MB)
- Use JPG for photos, PNG for graphics
- Recommended dimensions: 1920x1080 for banners, 800x1200 for portraits

## üåç Translations

All content can be translated to English, Ukrainian, and Russian.

**To add translations:**
1. Open any content item
2. Use the language tabs (EN | UK | RU) at the top
3. Enter content for each language
4. Save changes

**Note:** The slug/URL should remain the same across languages.

## ‚úÖ Publishing Workflow

### Draft vs Published

- **Draft** - Content is saved but not visible on the website
- **Published** - Content is live and visible to visitors

**To publish:**
1. Edit the item
2. Change **Status** to "Published"
3. Click **Save**

---

## üîÑ Common Tasks

### Reorder Featured Items
Featured items appear on the homepage in the order they were created. To change order:
1. Set items you don't want first to "not featured"
2. Create/update the item you want first as "featured"
3. Repeat for desired order

### Bulk Updates
Currently, items must be updated individually. For bulk changes, contact your developer.

### Deleting Content
1. Open the item
2. Click the **Delete** button
3. Confirm deletion

‚ö†Ô∏è **Warning:** Deleted content cannot be recovered.
