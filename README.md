# PURITY Fashion Studio

## Overview
PURITY Fashion Studio is a premium minimalist e-commerce platform for styling services and atelier. The platform is designed with a "Net-a-Porter" aesthetic, focusing on clean lines, high-quality imagery, and a seamless user experience.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Library:** Shadcn/UI
- **CMS:** Payload CMS (Planned)
- **Payments:** LiqPay + Stripe (Planned)

## Key Features
- Trilingual support (RU/UA/EN)
- Minimalist design system
- E-commerce functionality for services and products
- Headless CMS integration
